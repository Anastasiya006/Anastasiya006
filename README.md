## 🚀 stuff i've made

### 🎵 [PCB LED Audio Visualizer](https://github.com/Anastasiya006/PCB-LED-Audio-Visualizer)
LEDs that dance to music, on a circuit board i designed myself. yes it actually works. yes i'm proud of it.

### 🐾 [Byte Buddy](https://github.com/Anastasiya006/Byte-Buddy)
a little virtual pet that lives on your desktop and judges you while you code. built in Python and honestly kind of adorable.

### 🎮 [Jagame](https://github.com/Anastasiya006/Jagame)
a Java graphics library for beginner game devs — because raw Java graphics are a nightmare and nobody should have to deal with that alone.

### 🃏 [Poke-Pop](https://github.com/Anastasiya006/Poke-Pop)
a 2D turn-based Pokémon fighting game built in Java. the classic "i'll just make a quick game" project that turned into something real.

### 🌍 [ElectriumMap](https://github.com/Anastasiya006/electriumap)
contributed to an open-source EV mobility map for [Electrium](https://github.com/Electrium-Mobility) — TypeScript, real users, real impact.
